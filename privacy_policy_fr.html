GarminWO
Politique de confidentialité
Dernière mise à jour : 15 avril 2026  ·  Version extension : 1.4.2  ·  Voir sur GitHub
🔒 En résumé : GarminWO ne collecte aucune donnée personnelle, ne dispose d'aucun serveur, et ne vend ni ne partage vos informations.

1. Présentation

GarminWO est une extension Chrome gratuite qui importe des entraînements running dans Garmin Connect en analysant des photos de séances ou du texte grâce à une IA (Google Gemini ou Anthropic Claude). Cette politique explique quelles données sont traitées, comment elles sont utilisées et quels sont vos droits.

⚡ GarminWO n'a aucun serveur — tout le traitement s'effectue dans votre navigateur

2. Données traitées

2.1 Clés API

  • Quoi : Votre clé API Gemini (Google) ou Claude (Anthropic), saisie volontairement dans le popup de l'extension.
  • Où stockée : Uniquement en local sur votre appareil, via l'API Chrome chrome.storage.local.
  • Jamais : Transmise à un serveur GarminWO (il n'en existe pas), partagée avec des tiers, ni stockée hors de votre navigateur.

2.2 Images

  • Quoi : Photos ou captures d'écran que vous fournissez pour l'analyse, y compris les images capturées via le clic droit sur n'importe quelle page web.
  • Comment utilisées : Converties en Base64 et envoyées directement depuis votre navigateur vers l'API IA sélectionnée (Gemini ou Claude), uniquement pour l'analyse de la séance.
  • Jamais : Stockées au-delà de la durée de l'analyse, conservées par GarminWO, ni envoyées à un serveur GarminWO.
  • Traitement tiers : Les images sont traitées par Google ou Anthropic selon leurs politiques de confidentialité respectives.

2.3 Texte sélectionné

  • Quoi : Texte que vous sélectionnez sur une page web et envoyez à GarminWO via le menu contextuel (clic droit).
  • Comment utilisé : Envoyé directement à l'API IA sélectionnée pour l'analyse du plan d'entraînement.
  • Jamais : Stocké, journalisé, ni transmis ailleurs qu'à l'API IA.

2.4 Session Garmin Connect

  • Quoi : GarminWO utilise votre session Garmin Connect existante dans le navigateur (cookies) pour authentifier les appels API d'import et de suppression d'entraînements.
  • Comment utilisée : Les cookies sont lus depuis le contexte de la page Garmin Connect pour authentifier les requêtes vers l'API Garmin Connect uniquement.
  • Jamais : Les cookies ou identifiants ne sont stockés par GarminWO ni transmis en dehors des endpoints de l'API Garmin Connect.

4. Services tiers

GarminWO s'intègre aux services suivants. Leur utilisation est régie par leurs propres politiques de confidentialité :

  • Google Gemini API — policies.google.com/privacy
  • Anthropic Claude API — anthropic.com/privacy
  • Garmin Connect — garmin.com politique de confidentialité

5. Conservation des données

  • Clés API : Stockées localement jusqu'à ce que vous les effaciez manuellement ou désinstalliez l'extension.
  • Images et texte : Non stockés. Traités en mémoire uniquement, supprimés immédiatement après l'analyse.
  • Image/texte en attente (clic droit) : Stockés temporairement dans chrome.storage.local et supprimés immédiatement après récupération (usage unique).

6. Publicité et traçage

GarminWO ne contient aucune publicité, aucune analytique, aucune télémétrie et aucun traçage d'aucune sorte. Le développeur n'a accès à aucune donnée que vous traitez avec GarminWO.

7. Protection des mineurs

GarminWO ne collecte sciemment aucune information auprès d'enfants de moins de 13 ans. L'extension est destinée aux adultes gérant leurs propres données fitness.

8. Modifications de cette politique

En cas de mise à jour, la date "Dernière mise à jour" en haut de ce document sera révisée. Les modifications importantes seront signalées dans le changelog de l'extension sur GitHub.

9. Contact

Pour toute question concernant cette politique, veuillez ouvrir une issue sur le dépôt GitHub.
