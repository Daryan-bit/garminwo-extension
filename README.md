# garminwo-extension
An extension capable of importing workouts into Garmin Connect from an image or a screenshot under  Google Chrome.

Cette extension permet d'importer depuis une photo, une image ou même une sélection de texte des séances d'entrainements dans Garmin Connect sous  Google Chrome. Le transfert de la séance s'effectue ensuite vers la montre au travers du cable USB relié à l'ordinateur ou alors depuis l'application mobile Garmin Connect.
