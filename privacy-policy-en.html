Privacy Policy

Last updated: April 2026

🔒 GarminWO does not collect, store or transmit any personal data to third-party servers. Everything stays in your browser.

1. Data collected

GarminWO Extension only collects:

  • Anthropic API Key: stored locally in the browser via chrome.storage.local. It is never transmitted to any server other than Anthropic (api.anthropic.com) for image analysis.
  • Uploaded images: sent directly from your browser to the Anthropic API for analysis. They do not transit through any intermediate server.

2. Data not collected

The extension does not collect and has no access to:

  • Garmin Connect credentials (managed by your active session)
  • Activity or health data
  • Browsing history
  • Personal information (name, email, location)
  • Payment information

3. Third-party services

Anthropic API (Claude) — Image analysis is performed via the Anthropic API. Your API key and images are transmitted directly from your browser to Anthropic. See Anthropic's privacy policy: anthropic.com/legal/privacy

Garmin Connect — Workout imports use your active Garmin Connect session. No credentials are stored by the extension. See Garmin's privacy policy: garmin.com/privacy

4. Local storage

The Anthropic API key is stored in chrome.storage.local — an isolated storage space, specific to your browser and accessible only by this extension. It can be deleted at any time via the extension popup or by uninstalling the extension.

5. Permissions used

  • storage: to save the API key locally
  • activeTab: to interact with the active Garmin Connect page
  • host_permissions connect.garmin.com: to inject the button and call the workout API
  • host_permissions api.anthropic.com: to send images for analysis

6. Security

All communications with Anthropic and Garmin Connect use HTTPS. The API key is never displayed in plain text in the interface (password-type field).

7. Changes

This policy may be updated as the extension evolves. The last modification date is shown at the top of this page.
