Politique de confidentialité

Dernière mise à jour : avril 2026

🔒 GarminWO ne collecte, ne stocke et ne transmet aucune donnée personnelle à des serveurs tiers. Tout reste sur votre navigateur.

1. Données collectées

GarminWO Extension collecte uniquement :

  • Clé API Anthropic : stockée localement dans le navigateur via chrome.storage.local. Elle n'est jamais transmise à un serveur autre qu'Anthropic (api.anthropic.com) pour l'analyse d'image.
  • Images téléchargées : envoyées directement depuis votre navigateur vers l'API Anthropic pour analyse. Elles ne transitent par aucun serveur intermédiaire.

2. Données non collectées

L'extension ne collecte pas et n'a pas accès à :

  • Identifiants Garmin Connect (gérés par votre session active)
  • Données d'activité ou de santé
  • Historique de navigation
  • Informations personnelles (nom, email, localisation)
  • Données de paiement

3. Services tiers utilisés

API Anthropic (Claude) — L'analyse des images est effectuée via l'API Anthropic. Votre clé API et les images sont transmises directement depuis votre navigateur vers Anthropic. Consultez la politique de confidentialité d'Anthropic : anthropic.com/legal/privacy

Garmin Connect — L'import d'entraînements utilise votre session Garmin Connect active. Aucun identifiant n'est stocké par l'extension. Consultez la politique de confidentialité de Garmin : garmin.com/privacy

4. Stockage local

La clé API Anthropic est stockée dans chrome.storage.local — un espace de stockage isolé, propre à votre navigateur et accessible uniquement par cette extension. Elle peut être supprimée à tout moment via le popup de l'extension (case "Effacer les identifiants") ou en désinstallant l'extension.

5. Permissions utilisées

  • storage : pour sauvegarder la clé API localement
  • activeTab : pour interagir avec la page Garmin Connect active
  • host_permissions connect.garmin.com : pour injecter le bouton et appeler l'API workout
  • host_permissions api.anthropic.com : pour envoyer les images à analyser

6. Sécurité

Toutes les communications avec Anthropic et Garmin Connect utilisent HTTPS. La clé API n'est jamais affichée en clair dans l'interface (champ de type password).

7. Modifications

Cette politique peut être mise à jour en cas d'évolution de l'extension. La date de dernière modification est indiquée en haut de cette page.
